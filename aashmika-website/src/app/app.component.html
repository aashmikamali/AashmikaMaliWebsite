Claw Machine Bot - Dec 2018

The following three videos explain my latest project, the Claw Machine Bot. This project was the culmination of first year, first term Mechatronics Engineering at Waterloo, and was a creative way to make a machine or system comprising of mechatronic components. It was done through the use of Lego Mindstorms and other materials to build the complete system.
• Designed, built, and programmed a robot capable of competing against a user by collecting objects of a certain colour
• Created algorithm in C to pick and place a coloured object using EV3 motors, sensors and controllers

Free Play Mode

This explains and goes through the freeplay mode for the Claw Machine Bot. This mode simulates traditional Claw Machine Gamelplay, in which the user has a certain amount of time to move the claw wherever on the playing field, then drop with the press of a button.

Autonomous Mode

This explains and goes through the autonomous mode for the Claw Machine Bot. This mode elevates the traditional claw machine by providing an extra challenge of selecting a ball of the correct colour in a short amount of time, whilst playing against the machine!

Sumo Robot Challenge - Nov 2017

Robotics 12 Final isp: Designing and assembling a sumo bot that would push out other robots in a ring, based on the use of ultrasonic distance sensors, and IR sensors to detect the lines around the ring.
• Created an Arduino based Sumo-bot to compete with other Sumo-bots
• Used ultrasonic sensors to detect other robots and IR sensors to sense the outer ring
• Optimized mechanical robot design by implementing a conical skirt for defence and PWM drive for optimal position control
My robot is the white robot with the circular base, that remains the last one standing!

Autonomous Line Tracing Robot - Jan 2017

Computer engineering 12: Designed, assembled and programmed a robot to achieve functionality. Implemented and fine-tuned control algorithm to increase speed.
• Created an Arduino based robot that completed a circuit in the shortest time amongst peers
• Implemented line detection using analog values read from LDRs coupled with LEDs
• Devised control scheme in Arduino code to maximise speed using analog data from three LDR/LED pairs

Traffic Light System - Jun 2016

Computer engineering 11: Designed, assembled and programmed a realistic miniaturized traffic light intersection using Arduino Uno.
• Created a realistic traffic light system using servos, LDRs, IR sensors and LEDs, controlled by an Arduino
• Filtered ADC inputs by averaging, used PWM to control dimming of street lights/LEDs and to ramp up/down car speed
• Implemented a state machine for system control and polled for pedestrian cross-walk button

Whack-a-mole - Jan 2015
