Project Descriptions and Videos

MowAI - Robotic Platform - Dec 2018

MowAI is my newest project that incorporates the design of my motor control circuitry, using the STMF320 microcontroller. The goal of this project is to have lawn mowing robot that can be remote controlled and have obstacle detection abilities as well as motors to change the direction and location of the bot. Each wheel of the bot will have 2 motors, an encoder and a potentiometer, an accelerometer and gyroscope, and 2 TOF sensors. The PCB for this project is displayed at the top of the page! The following are some of the tasks I have completed with regards to MowAI.
• Schematic design and layout of board assembly incorporating embedded servo motor control and H-bridge PWM control for motion using STM32F0 microcontroller
• Developed code to implement servo control using potentiometer feedback, I2C and SPI interface to MEMS sensors
• Power supply design and layout for voltage rails with low quiescent current for battery applications using WEBENCH

Midnight Sun Boards: Pedal Control Interface & Power Distribution Boards

Top/Bottom-Power Distribution



This board deals with distributing power to all of the low voltage systems (12 volt systems) in the solar car. This includes the driver controls, the lights, the horn and even the battery management system.

Top/Bottom-Pedal Interface



The pedal board receives inputs from the angle of the brake and acceleration pedals based on the turn of a potentiometer placed within. They also have active low pass filters to make sure the received signal is uninterrupted by the noise caused from the motors and radio communications.

Top/Bottom-2d-DCDC Battery Pack converter





Top/Bottom-3d-DCDC Battery Pack converter


This project takes the 115V nominal power coming from the battery, and uses a DCM dc-dc converter to supply 13.5 volts trimmed with about 16A out. This 16A is then sent to power distribution and provides all the power for the low voltage systems in the car. Common mode filters (including using y capacitor), common mode chokes, effective bulk capacitor layout and high current rating shank terminals were all used for an effective design.
