Claw Machine Bot - Dec 2018

The following three videos explain my latest project, the Claw Machine Bot. This project was the culmination of first year, first term Mechatronics Engineering at Waterloo, and was a creative way to make a machine or system comprising of mechatronic components. It was done through the use of Lego Mindstorms and other materials to build the complete system.

Free Play Mode

This explains and goes through the freeplay mode for the Claw Machine Bot. This mode simulates traditional Claw Machine Gamelplay, in which the user has a certain amount of time to move the claw wherever on the playing field, then drop with the press of a button.

Autonomous Mode

This explains and goes through the autonomous mode for the Claw Machine Bot. This mode elevates the traditional claw machine by providing an extra challenge of selecting a ball of the correct colour in a short amount of time, whilst playing against the machine!
